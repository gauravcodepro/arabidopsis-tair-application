Introduction:

This application is aimed for development of providing you a interface to the TAIR, so that you can access the TAIR with in the desktop application

  • Allows for all the embedded analysis from TAIR such as:

    • Analysis of the genes and the proteins

    • Gene ID conversion

    • Analyzing Ontologies

    • Gene and Protein Sequence Analyzer

    • Extracting the genome sub-fractions

    • gene, proteins to NCBI and KEGG mappings
